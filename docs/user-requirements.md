# 사용자 요구사항 (User Requirements)

- 프로젝트: Addinedu 4기 최종 프로젝트 — 로봇 호텔
- 기간: 2026-04-28 ~ 2026-06-04
- 작성일: 2026-04-28

> Priority 표기: **High** = 구현 대상 / **Low** = 구현 보류(시나리오 완결성 위해 기록)

## 1. 예약 / 도착 전

| UR-ID | Name | Description | Priority |
|---|---|---|---|
| UR-001 | 예약 등록 | 직원이 PyQt5 직원 앱에서 손님 예약(성명, 연락처, 객실 유형, 투숙 기간)을 등록한다. | Low |
| UR-002 | QR 발급 | 손님이 예약 시 예약번호가 인코딩된 QR 코드를 발급받는다. | Low |

## 2. 체크인 (OpenArm 응대)

| UR-ID | Name | Description | Priority |
|---|---|---|---|
| UR-003 | 예약 조회 | 손님이 체크인 키오스크(노트북)에 QR 코드를 제시하면, 키오스크가 QR을 인식해 예약을 조회한다. | High |
| UR-004 | 신원 확인 | 시스템이 QR에서 추출한 예약번호로 예약 데이터를 매칭해 신원을 확인한다. | High |
| UR-005 | 결제 처리 | 손님이 체크인 시 카드 결제로 투숙 비용을 정산한다. | High |
| UR-006 | 카드키 전달 | OpenArm이 카드키를 집어 손님에게 전달한다. | High |
| UR-007 | 어메니티 전달 | OpenArm이 웰컴 키트(생수 등 소형 어메니티)를 집어 손님에게 전달한다. | High |
| UR-008 | 환영 인사 모션 | OpenArm이 환영 인사 동작을 수행한다. | High |
| UR-009 | 체크인 완료 안내 | 체크인 완료 후 손님 휴대폰에 객실 번호와 위치 안내가 표시된다. | High |

## 3. 객실 내 서비스 (OMX)

| UR-ID | Name | Description | Priority |
|---|---|---|---|
| UR-010 | 음성 조명 제어 | 손님이 휴대폰 음성 명령("불 켜줘")으로 OMX를 호출하면, OMX가 객실의 물리적 조명 스위치를 직접 누른다. | High |
| UR-011 | 신발 정리 | 손님이 음성 명령으로 신발 정리를 요청하면, OMX가 슬리퍼를 인식해 신발장 위치로 옮겨 정리한다. | High |
| UR-012 | 책상 정리 | 손님이 음성 명령으로 책상 정리를 요청하면, OMX가 책상 위 물체를 인식해 지정된 위치로 정리한다. | High |
| UR-013 | 모닝콜 설정 | 손님이 휴대폰 웹앱에서 모닝콜 시각을 설정한다. | High |
| UR-014 | 모닝콜 수행 | 설정된 시각에 OMX가 모닝콜 동작을 수행하고, 손님 휴대폰에도 알림이 전송된다. | High |

> 주: OMX 3대와 작업 4종(UR-010~014)의 매핑은 추후 결정한다.

## 4. 룸서비스 (Pinky 배달)

| UR-ID | Name | Description | Priority |
|---|---|---|---|
| UR-015 | 룸서비스 주문 | 손님이 휴대폰 웹앱에서 룸서비스 메뉴를 선택해 주문한다. | High |
| UR-016 | 주문 적재 | 직원이 주문된 물품을 Pinky의 적재함에 싣는다. | High |
| UR-017 | 자율 배달 주행 | Pinky가 주문 객실까지 자율 주행으로 이동한다. | High |
| UR-018 | 도착 알림 | Pinky가 객실 도착 시 손님 휴대폰에 도착 알림을 보낸다. | High |
| UR-019 | 물품 수령 | 손님이 Pinky의 적재함에서 물품을 직접 수령한다. | High |
| UR-020 | 자동 복귀 | Pinky가 배달 완료 후 대기 위치/충전소로 자동 복귀한다. | High |

## 6. 직원 운영 (PyQt5 데스크톱 앱)

| UR-ID | Name | Description | Priority |
|---|---|---|---|
| UR-022 | 객실 상태 보드 | 직원이 객실 점유 현황(공실/투숙/청소 중)을 한 화면에서 확인한다. | High |
| UR-023 | 로봇 상태 모니터링 | 직원이 OpenArm·Pinky·OMX의 위치, 배터리, 현재 작업 상태를 실시간으로 확인한다. | High |
| UR-024 | 작업 큐 확인 | 직원이 룸서비스 주문 및 객실 작업 요청 큐를 확인한다. | High |
| UR-025 | 수동 제어/오버라이드 | 직원이 로봇 작업을 수동으로 시작하거나 중단한다. | High |

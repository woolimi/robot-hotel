# User Requirements

# 사용자 요구사항 (User Requirements)

## 1. 예약 / 도착 전

| UR-ID | Name | Description | Priority |
| --- | --- | --- | --- |
| UR-001 | 예약 등록 | 손님은 성명, 연락처, 객실 유형, 투숙 기간을 입력하여 예약을 할 수 있다. | Low |
| UR-002 | 결제 처리 | 손님은 예약시에 카드 결제로 투숙 비용을 정산한다. | Low |
| UR-003 | QR 발급 | 손님이 예약 완료 후 예약번호가 인코딩된 QR 코드를 발급받는다. | Low |

## 2. 체크인

| UR-ID | Name | Description | Priority |
| --- | --- | --- | --- |
| UR-004 | 예약 조회 | 손님이 체크인 키오스크에 QR 코드를 제시하면, 키오스크가 QR을 인식해 예약을 조회한다. | High |
| UR-005 | 신원 확인 | 시스템이 QR에서 추출한 예약번호로 예약 데이터를 매칭해 신원을 확인한다. | High |
| UR-006 | 카드키 전달 | 카운터 로봇은 카드키를 집어 손님에게 전달한다. | High |
| UR-007 | 어메니티 전달 | 카운터 로봇은 웰컴 키트를 집어 손님에게 전달한다. | High |
| UR-008 | 환영 인사 모션 | 카운터 로봇은 환영 인사 동작을 수행한다. | High |
| UR-009 | 체크인 완료 안내 | 체크인 완료 후 시스템에 객실 번호와 위치 안내가 표시된다. | High |
| UR-010 | 객실 안내 | 체크인 완료 후 안내 로봇은 손님을 해당 객실까지 안내한다. | High |

## 3. 객실 내 서비스

| UR-ID | Name | Description | Priority |
| --- | --- | --- | --- |
| UR-011 | 음성 조명 제어 | 손님은 음성 명령으로 점등/소등을 요청하면 객실 로봇은 불을 켜고 끈다. | High |
| UR-012 | 신발 정리 | 손님은 음성 명령으로 신발 정리를 요청하면, 객실 로봇은 신발을 가지런히 정리한다. | High |
| UR-013 | 책상 정리 | 손님은 음성 명령으로 책상 정리를 요청하면, 책상 위 물체를 인식해 지정된 위치로 정리한다. | High |
| UR-014 | 모닝콜 설정 | 손님은 휴대폰 웹앱에서 모닝콜 시각을 설정할 수 있다. | High |
| UR-015 | 모닝콜 수행 | 모닝콜이 설정된 시각에 객실 로봇은 모닝콜 동작을 수행한다. | High |

## 4. 룸서비스

| UR-ID | Name | Description | Priority |
| --- | --- | --- | --- |
| UR-016 | 룸서비스 주문 | 손님은 시스템에 룸서비스 메뉴를 선택해 주문한다. | High |
| UR-017 | 주문 적재 | 직원은 주문된 물품을 배달 로봇의 적재함에 싣는다. | High |
| UR-018 | 자율 배달 주행 | 배달 로봇은 주문 객실까지 자율 주행으로 이동한다. | High |
| UR-019 | 도착 알림 | 배달 로봇은 객실 도착 시 손님 휴대폰에 도착 알림을 보낸다. | High |
| UR-020 | 물품 수령 | 손님은 배달 로봇의 적재함에서 물품을 직접 수령한다. | High |
| UR-021 | 자동 복귀 | 배달 로봇은 배달 완료 후 대기 위치/충전소로 자동 복귀한다. | High |

## 5. 체크아웃

| UR-ID | Name | Description | Priority |
| --- | --- | --- | --- |
| UR-022 | 체크아웃 요청 | 손님은 체크인 데스크 키오스크에서 카드키를 인식시켜 체크아웃을 시작한다. | Low |
| UR-023 | 카드키 반납 | 손님은 체크인 데스크의 카드키 반납함에 카드키를 넣는다. | Low |
| UR-024 | 카드키 회수 | 카운터 로봇은 반납함에서 카드키를 집어 보관함에 정리한다. | Low |
| UR-025 | 체크아웃 완료 안내 | 시스템은 카드키 회수가 확인되면 키오스크에 체크아웃 완료를 안내하고, 해당 객실 상태를 '청소 대기'로 전환한다. | Low |

## 6. 직원 운영

| UR-ID | Name | Description | Priority |
| --- | --- | --- | --- |
| UR-026 | 객실 상태 보드 | 직원이 객실 점유 현황(공실/투숙/청소 중)을 한 화면에서 확인한다. | High |
| UR-027 | 로봇 상태 모니터링 | 직원이 OpenArm·Pinky·OMX의 위치, 배터리, 현재 작업 상태를 실시간으로 확인한다. | High |
| UR-028 | 작업 큐 확인 | 직원이 룸서비스 주문 및 객실 작업 요청 큐를 확인한다. | High |
| UR-029 | 수동 제어/오버라이드 | 직원이 로봇 작업을 수동으로 시작하거나 중단한다. | High |

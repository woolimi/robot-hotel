# 사용자 요구사항 (User Requirements)

## 1. 등록

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-001 | 자녀·학부모 등록 | 교사는 언제든 자녀를 등록하고 연결된 학부모 정보를 시스템에 등록할 수 있어야 한다. |
| UR-002 | 학부모 시스템 접근 | 학부모는 등록 후 발급된 계정으로 시스템에 접근할 수 있어야 한다. |

## 2. 등원

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-003 | 아이 등원 응대 | 아이는 등원 시 자동으로 출석 처리되고 이름과 함께 환영받을 수 있어야 한다. |
| UR-004 | 학부모 등원 확인 | 학부모는 자녀의 등원 여부를 시스템을 통해 알 수 있어야 한다. |
| UR-005 | 교사 등원 기록 | 교사는 아이의 등원 시각과 출결 현황을 자동으로 확인할 수 있어야 한다. |

## 3. 레크리에이션

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-006 | 놀이 진행 | 아이는 시스템과 함께 다양한 놀이를 즐길 수 있어야 한다. |

## 4. 운반 보조

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-007 | 짐 운반 보조 | 교사는 짐 운반에 시스템의 도움을 받을 수 있어야 한다. |

## 5. 정리 보조

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-008 | 장난감 정리 | 교사는 레크리에이션 후 장난감을 바구니에 정리하는 데 시스템의 도움을 받을 수 있어야 한다. |

## 6. 낮잠

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-009 | 낮잠 환경 조성 | 아이는 낮잠 시간에 적합한 환경(자장가·소등)에서 휴식할 수 있어야 한다. |
| UR-010 | 낮잠 제어 | 교사는 낮잠 모드를 시작·종료할 수 있어야 한다. |

## 7. 하원

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-011 | 아이 하원 응대 | 아이는 하원 시 자동으로 출석 처리되고 이름과 함께 작별 인사를 받을 수 있어야 한다. |
| UR-012 | 학부모 하원 확인 | 학부모는 자녀의 하원 여부를 시스템을 통해 알 수 있어야 한다. |
| UR-013 | 교사 하원 기록 | 교사는 아이의 하원 시각과 출결 현황을 자동으로 확인할 수 있어야 한다. |

## 8. 로봇 인터랙션

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-014 | 자연어 명령 | 아이와 교사는 모든 로봇에 음성 명령으로 동작을 지시할 수 있어야 한다. |
| UR-015 | 음성 모드 전환 | 교사는 모든 로봇에 음성으로 시스템 모드 전환을 지시할 수 있어야 한다. |
| UR-016 | 로봇 표정 | 아이는 로봇 디스플레이의 표정을 통해 로봇의 상태를 인지하고 친근감을 느낄 수 있어야 한다. |

## 9. 교사 운영

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-017 | 정보 조회 | 교사는 자녀·학부모 정보·점심메뉴·자녀 일과 보고서를 시스템에서 조회할 수 있어야 한다. |

## 10. 학부모 시스템

| UR-ID | Name | Description |
| --- | --- | --- |
| UR-018 | 점심메뉴 조회 | 학부모는 일별 점심메뉴를 확인할 수 있어야 한다. |
| UR-019 | 사진첩 조회 | 학부모는 자녀가 포함된 일상 사진을 사진첩에서 조회할 수 있어야 한다. |
| UR-020 | 일과 보고서 조회 | 학부모는 자녀의 하루 활동·감정을 정리한 일과 보고서를 하원 시점에 조회할 수 있어야 한다. |
